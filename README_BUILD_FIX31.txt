KTin buildfix31

변경 사항:
1. 갈무리 보기 창 자동 붙기(스냅) 추가
   - 갈무리창을 메인창/다른 갈무리창 가까이 가져간 뒤 마우스를 놓으면 자동으로 붙습니다.
   - 위/아래로 붙을 때는 대상 창의 가로폭에 맞춥니다.
   - 모니터 작업 영역을 벗어나면 세로 크기를 줄입니다.
   - Shift 키를 누른 채 이동하면 자동 붙기를 임시로 무시합니다.

2. 항상 위 옵션 추가
   - 옵션 > 환경설정 > 기타 설정: 메인창 항상 위
   - 갈무리 보기 창 메뉴/우클릭 메뉴: 항상 위

3. 주소록 새 주소/수정 창 높이 보정
   - '접속 후 60초...' 안내와 확인/취소 버튼 사이의 과도한 빈 공간을 줄였습니다.

4. 접속 유지 상태 판정 보강
   - TinTin++의 접속 성공/끊김 문구 감지 tick을 추가했습니다.
   - isConnected/isSessionActive가 흔들려도 마지막 접속 성공 이후 끊김 감지가 없으면 접속유지가 계속 동작하도록 보강했습니다.
   - 새 접속 시도 시 이전 접속 성공 상태를 초기화합니다.

5. 갈무리 보기 파일 추적 보강
   - 갈무리 파일이 새로 생성/변경되면 이미 열린 갈무리 보기 창도 현재 갈무리 파일을 따라가도록 수정했습니다.
