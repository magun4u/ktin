KTin buildfix25
================

1. 갈무리 로그 저장 방식을 원본 ANSI 바이트 저장으로 변경했습니다.
   - 본체 화면 렌더링 경로에서 패턴 검색은 하지 않습니다.
   - 갈무리 보기의 필터 검색은 ANSI 제거 텍스트 기준으로 합니다.

2. 갈무리 필터 설정에 ANSI 보기 체크박스를 추가했습니다.
   - 전체/잡담/대화/경매/아이템 획득/경험치/사용자1~3별로 선택 가능합니다.
   - ANSI 보기 ON: 검정 배경 + 현재 메인 ANSI 테마 기준으로 색상 표시합니다.
   - ANSI 보기 OFF: ANSI 코드를 제거한 일반 텍스트로 표시합니다.

3. 보기 > 갈무리 메뉴에 갈무리창 모두 닫기 메뉴를 추가했습니다.
   - 갈무리 보기 창이 1개 이상 열려 있을 때만 메뉴에 표시됩니다.
   - 갈무리 보기 창이 없으면 메뉴에서 숨겨집니다.

주의:
- buildfix25부터 새로 저장되는 갈무리 로그는 ANSI 원문을 포함합니다.
- 이전 빌드에서 만든 일반 텍스트 로그도 갈무리 보기에서 계속 읽을 수 있습니다.
