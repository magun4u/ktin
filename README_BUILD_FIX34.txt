주요 수정:
1. 갈무리 보기 ANSI RichEdit 색상 출력 안정화
   - 장시간 테스트에서 ANSI 보기 켜짐 상태가 UI 멈춤/검정 글자/메뉴 무응답 원인으로 의심되어 일단 비활성화.
   - 갈무리 로그 파일에는 ANSI 원문이 계속 저장됨.
   - 갈무리 보기창은 ANSI 코드를 제거한 평문으로 표시.
   - 필터 설정의 ANSI 보기 체크박스는 호환을 위해 남기되 회색 처리.

2. 갈무리 보기 공통 tail 타이머
   - 갈무리 보기창마다 별도 타이머를 돌리지 않고 대표 창 하나의 타이머가 모든 갈무리창을 순서대로 갱신.
   - 각 창 처리 중 재진입 방지 플래그 추가.

3. 갈무리 보기 상태바 줄 수 보정
   - 표시 버퍼는 2000줄까지만 유지하지만, 상태바의 줄 카운터는 누적 매칭 줄 수를 표시.
   - 예: 4026줄 | 표시 2000줄

4. 갈무리 보기 상태바 폰트 변경
   - 상태바는 메인 로그 폰트가 아니라 Windows 메뉴/대화상자 계열 폰트를 사용.

5. 파일 메뉴에 새 창 띄우기 추가
   - 현재 KTin 실행 파일을 한 번 더 실행해 새 프로그램 창을 띄움.
