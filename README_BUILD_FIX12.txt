KTin safe buildfix12
=====================

추가 기능:
- 보기 > 갈무리 > 전체 수신 로그 저장
  화면 캡처/정규식 분류 없이 TinTin++에서 수신된 텍스트를 UTF-8 로그 파일로 저장합니다.
  저장 위치: 실행 폴더\log\YYYYMMDDHHMMSS.txt
- 보기 > 갈무리 > 갈무리 보기 > 전체/잡담/경매/아이템 획득/사용자 문자열
  저장된 로그 파일을 1초 간격으로 tail 방식으로 읽어 새로 추가된 줄만 보여줍니다.
  KTin 본체의 화면 렌더링과 분리되어 있으며, Tail 창은 표시 줄 수를 제한합니다.
- 보기 > 갈무리 > 갈무리 폴더 열기

주의:
- Tail 필터는 본체 수신/렌더링 경로에서 실행하지 않습니다.
- 사용자 문자열은 정규식이 아니라 단순 포함 검색입니다.
- 정규식 기반 트리거/채팅캡처는 계속 제거 상태입니다.
