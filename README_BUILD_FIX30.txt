Buildfix30 - 터미널 화면 여백 설정 추가

변경 사항:
- 옵션 → 환경설정 → 일반 설정에 화면 여백 설정(px) 추가
  왼쪽 / 오른쪽 / 위 / 아래
- 기본값은 모두 0px
- 화면 정렬은 설정된 여백을 제외한 영역 안에서 계산됨
  왼쪽: 왼쪽 여백부터 시작
  중앙: 왼쪽/오른쪽 여백을 제외한 영역의 중앙
  오른쪽: 오른쪽 여백을 남기고 붙임
- 출력 높이가 화면보다 클 때는 아래 여백을 보존하고 프롬프트가 보이도록 아래쪽 기준으로 정렬
- 보기 → 화면 여백 없애기에서 창 크기를 맞출 때도 설정된 여백을 포함해 계산
