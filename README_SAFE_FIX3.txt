KTin 안전판 buildfix3
=====================

이번 수정 내용:
1. 상단 메뉴가 보이지 않던 문제 수정
   - MainWndProc WM_PAINT에서 DrawCustomMenuBar()를 다시 호출합니다.
   - 상단 메뉴 클릭으로 드롭다운 메뉴가 열리도록 MainWndProc에 마우스 처리 추가.

2. 메뉴 숨김 상태 복구
   - 실행 직후 menuHidden=false로 강제하여 기본은 상단 메뉴 표시.
   - 메뉴 숨김 상태에서 우클릭하면 "상단 메뉴 보이기" 메뉴가 표시됩니다.
   - 터미널 우클릭 메뉴는 ownerdraw 대신 일반 문자열 메뉴로 변경하여 표시 안정성을 높였습니다.

3. 입력창 전송 안 됨 수정
   - isConnected 값이 false로 남아도 입력 명령을 TinTin++ 프로세스로 항상 보냅니다.
   - #명령뿐 아니라 일반 MUD 명령도 막지 않습니다.

4. 상태바 기본 표시 보정
   - 상태바 형식이 비어 있으면 "KTin 준비"를 기본값으로 넣습니다.
   - statusbar count 저장 버그를 수정했습니다.

5. 전각/박스/기호 폭 기본값 보정
   - 모호한 동아시아 문자 폭 기본값을 넓게 처리하도록 변경했습니다.
   - 박스문자, 도형, 화살표 등이 1칸으로 처리되어 깨져 보이는 문제를 줄입니다.

여전히 제거/비활성화한 기능:
- 실시간 트리거/하이라이트 매칭
- 채팅 캡처 패턴 엔진
- 채팅 캡처창 누적 출력
- 자동 갈무리 기본 활성화
