- EUC-KR/CP949 서버 지도 출력에서 ○, ─, │ 같은 모호폭 기호가 깨지는 문제를 보정했습니다.
- 전각 기호를 논리적으로 2칸으로 처리하되, 렌더링 시 +16 픽셀 고정 이동을 제거했습니다.
- 글리프가 2칸보다 좁으면 2칸 셀 안에서 가운데 정렬합니다.
- 모호폭 문자 목록에 수학기호, 원문자, Fullwidth ASCII 영역을 보강했습니다.

주의:
- EUC-KR 서버는 TinTin++에서 #CONFIG {CHARSET} {CP949TOUTF8}을 써야 합니다.
- 주소록/빠른연결에서 EUC-KR(CP949)을 고르면 이 명령은 자동 전송됩니다.
