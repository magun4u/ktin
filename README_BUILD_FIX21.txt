KTin buildfix21
================

변경 내용
1. 보기 메뉴 구조 정리
   - 보기 > 갈무리
       * 갈무리 켜기 / 갈무리 끄기
       * 갈무리 폴더 열기
   - 보기 > 갈무리 보기 를 상위 메뉴로 이동
       * 전체, 잡담, 경매, 대화, 아이템 획득, 경험치, 사용자1~3, 임시 문자열
   - 보기 > 갈무리 필터 설정... 을 상위 메뉴로 이동

2. 갈무리 켜기/끄기 메뉴명 동적 변경
   - 갈무리가 꺼져 있으면 "갈무리 켜기"
   - 갈무리가 켜져 있으면 "갈무리 끄기"

3. 갈무리 보기 timestamp
   - 전체 보기: 원문 그대로 표시
   - 잡담/대화/경매/아이템/경험치/사용자 정의/임시 문자열 보기:
     각 줄 앞에 [HH:MM:SS] timestamp 표시

4. 사용자 문자열 메뉴 이름 정리
   - "사용자 문자열..."은 저장되는 사용자4가 아니라 1회성 검색이므로
     "임시 문자열..."로 이름을 바꿈
   - 사용자 정의 저장 필터는 사용자1, 사용자2, 사용자3만 유지
