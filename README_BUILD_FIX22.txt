KTin buildfix22
================

변경 내용:
1. 보기 -> 갈무리 메뉴의 토글 문구를 "갈무리 켜짐" / "갈무리 꺼짐"으로 변경.
2. 갈무리 보기 창을 다중 창 방식으로 변경.
   - 잡담 창을 연 뒤 경매 창을 다시 열어도 기존 창이 닫히지 않습니다.
3. 갈무리 보기 창에 탭 컨트롤 추가.
   - 기본은 메뉴에서 선택한 항목 1개 탭만 표시합니다.
   - 갈무리 보기 창 우클릭 또는 창 메뉴의 보기 -> 탭 설정에서 표시할 탭을 선택할 수 있습니다.
   - 각 창마다 표시 탭을 다르게 둘 수 있습니다.
4. 갈무리 보기 창 메뉴 숨기기/보이기 지원.
   - 메뉴 숨김 상태에서는 우클릭 메뉴에 "메뉴 보이기"가 표시됩니다.
5. 갈무리 보기 창의 왼쪽 드래그 선택 복사 지원.
   - 선택 후 마우스 왼쪽 버튼을 놓으면 선택 내용이 클립보드로 복사됩니다.
   - 우클릭 메뉴의 선택/전체 복사도 지원합니다.
6. 갈무리 보기 창의 빈 마지막 줄 문제 완화.
   - 새 줄을 append할 때 끝에 CRLF를 붙이지 않고, 다음 줄 앞에 CRLF를 붙여 마지막 빈 줄 커서를 줄였습니다.

주의:
- Tail 필터링은 기존처럼 로그 파일의 새로 추가된 줄만 읽어 단순 포함 검색합니다.
- KTin 본체의 터미널 렌더링/수신 처리 경로에는 정규식 기반 실시간 캡처를 넣지 않았습니다.
