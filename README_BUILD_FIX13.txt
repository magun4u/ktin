================

수정 내용:
1. log_tail.cpp 315줄의 std::max/LONG/int 충돌 오류 수정.
2. 보기 > 갈무리 메뉴의 하위 메뉴 구성을 보강.
3. 갈무리 보기 필터를 사용자가 직접 설정할 수 있도록 추가.
   - 보기 > 갈무리 > 갈무리 필터 설정...
   - 잡담 / 경매 / 대화 / 아이템 획득 문자열을 세미콜론(;)으로 구분해서 설정.
   - 정규식은 사용하지 않음. 단순 포함 검색만 사용하여 안정성을 우선함.
   - 잡담 특수값:
     []:  => [이름]: 내용 형식
     <>:  => <이름>: 내용 형식
4. 보기 > 갈무리 > 갈무리 보기 > 대화 항목 추가.
