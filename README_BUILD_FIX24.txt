변경 내용:
1. 갈무리 보기 창의 하단 가로 스크롤바 제거
   - 갈무리 보기 출력 EDIT 컨트롤에서 WS_HSCROLL / ES_AUTOHSCROLL 제거.

2. 갈무리 보기 창 상태바 숨기기/보기 지원
   - 보기 메뉴에 상태바 숨기기 / 상태바 보기 항목 추가.
   - 상태바를 숨기면 편집 영역이 아래까지 확장됨.
   - 메뉴를 숨긴 상태에서도 우클릭 메뉴에서 상태바 숨기기/보기를 실행 가능.

3. 갈무리 보기 창 탭 자동 숨김
   - 표시 탭이 1개뿐이면 탭 컨트롤을 숨김.
   - 제목 표시줄에 이미 “갈무리 보기 - 잡담”처럼 표시되므로 단일 탭은 보이지 않게 함.
   - 탭 설정에서 2개 이상 선택하면 탭 컨트롤이 다시 나타남.
