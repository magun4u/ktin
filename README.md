# KTin

KTin은 TinTin++를 Windows에서 편하게 사용하기 위한 GUI 래퍼 프로그램입니다.  
기존 TinTin++의 강력한 스크립트 기능은 그대로 사용하면서, Windows용 입력창, 단축버튼, 주소록, 갈무리 보기, 로그 저장, 창 자동 붙기 같은 편의 기능을 제공합니다.

KTin은 MUD 접속용으로 제작되었으며, 한국어 MUD 환경과 ANSI 출력, CP949/EUC-KR 서버 사용을 고려해 개발되었습니다.

---

## 주요 기능

### TinTin++ GUI 래퍼

- TinTin++ 실행 파일을 KTin 내부에서 실행
- Windows GUI 창에서 TinTin++ 입출력 표시
- 한글 입력 지원
- ANSI 색상 출력 지원
- TinTin++ 스크립트 사용 가능
- `main.tin` 자동 실행 지원

---

### MUD 출력창

- 고정폭 폰트 기반 MUD 출력
- ANSI 색상 테마 지원
- 화면 가로칸 수 설정
- 화면 세로줄 수 설정
- 왼쪽 / 중앙 / 오른쪽 정렬
- 화면 여백 설정 지원

화면 여백 설정 예:

```text
왼쪽: 4
오른쪽: 4
위: 2
아래: 2
````

프레임에 글자가 너무 붙어 보일 때 사용할 수 있습니다.

---

### 입력창

* 하단 입력창 제공
* 명령 입력 후 TinTin++로 전송
* 빈 엔터 전송 지원
* 최근 입력 기록 확인
* 방향 버튼 및 단축버튼 지원

---

### 단축버튼

* 하단 단축버튼 제공
* 자주 쓰는 명령을 버튼에 등록 가능
* 예: `점수`, `상태`, `가방`, `장비`, `지도`, `북`, `남`, `동`, `서`

---

### 주소록

* 서버 주소와 포트 저장
* 주소록에서 바로 접속
* 자동 로그인 지원
* 아이디 패턴 / 비밀번호 패턴 설정
* 접속 후 60초 동안만 자동 로그인 패턴 검사
* 접속 실패 또는 끊김 시 자동 재접속 옵션 지원

---

### 접속 유지

* 일정 시간마다 지정 명령을 자동 전송
* 예: 5초마다 `봐` 전송
* 접속 상태일 때만 접속 유지 명령 전송
* 접속 중 / 로그인 중 상태 판정 보강

---

### 갈무리

KTin은 MUD 출력을 로그 파일로 저장하는 갈무리 기능을 제공합니다.

* 전체 출력 로그 저장
* 로그 파일 자동 생성
* 갈무리 켜짐 / 꺼짐 상태 표시
* 접속 중 생성된 로그 파일 추적
* 갈무리 보기창에서 필터별 보기 지원

로그 파일 예:

```text
D:\Tintin_gui\log\20260707235023.txt
```

---

### 갈무리 보기창

갈무리 보기창을 별도로 열어 필요한 문장만 확인할 수 있습니다.

지원 필터:

* 잡담
* 대화
* 경매
* 아이템 획득
* 경험치
* 사용자 정의 필터

지원 기능:

* 여러 갈무리 보기창 동시 실행
* 갈무리창별 탭 설정
* 단일 탭일 때 탭 숨김
* 우클릭 메뉴
* 마우스 드래그 복사
* 상태바 표시
* 현재 대상 로그 파일 표시
* 누적 줄 수 표시

상태바 예:

```text
4026줄 | 표시 2000줄 | D:\Tintin_gui\log\20260707235023.txt
```

표시 버퍼는 안정성을 위해 최대 2000줄을 유지합니다.
단, 상태바의 누적 줄 수는 실제 매칭된 전체 줄 수를 표시합니다.

---

### 갈무리 보기 안정성

KTin 2.6에서는 장시간 접속 안정성을 위해 갈무리 보기 구조를 보강했습니다.

* 여러 갈무리창이 같은 로그 파일을 동시에 추적할 때 부하 감소
* 공통 tail 타이머 사용
* 갈무리창 갱신 중 재진입 방지
* ANSI RichEdit 서식 적용으로 인한 멈춤 문제 방지를 위해 갈무리 보기 ANSI 색상 출력은 비활성화

로그 파일에는 ANSI 원문이 저장됩니다.
다만 갈무리 보기창에서는 안정성을 위해 ANSI 코드를 제거한 평문으로 표시합니다.

---

### 창 자동 붙기

갈무리 보기창을 메인창 가까이 가져가면 자동으로 붙는 기능을 제공합니다.

지원 기능:

* 갈무리창을 메인창 위 / 아래 / 왼쪽 / 오른쪽에 자동 붙이기
* 갈무리창끼리 자동 붙이기
* 메인창 위/아래에 붙을 때 가로폭 자동 맞춤
* 화면 밖으로 넘어갈 경우 크기 자동 보정
* 메인창 이동 시 붙어 있는 갈무리창도 함께 이동
* `Shift` 키를 누른 채 이동하면 자동 붙기 임시 무시

예:

```text
[갈무리창]
[갈무리창]
[메인창]
```

---

### 항상 위

* 메인창 항상 위 옵션
* 갈무리 보기창 항상 위 옵션
* 갈무리창별 항상 위 설정 가능

---

### 새 창 띄우기

KTin은 현재 멀티탭 방식이 아닙니다.
여러 서버나 여러 세션을 사용하려면 새 창을 띄워 사용할 수 있습니다.

```text
파일 → 새 창 띄우기
```

---

### Mud둥근모 폰트 지원

`MudDunggeunmo-Regular.ttf` 파일을 사용할 수 있습니다.

지원 방식:

1. 빌드 시 소스 폴더에 `MudDunggeunmo-Regular.ttf`가 있으면 실행 파일에 폰트 리소스로 포함
2. 실행 파일 옆에 `MudDunggeunmo-Regular.ttf`가 있으면 실행 중 프로세스 전용 폰트로 로드
3. Windows에 폰트를 설치하지 않아도 KTin에서 사용 가능

## 빌드 환경

권장 환경:

* Windows 10 / Windows 11
* Visual Studio 2022
* MSVC C++ 빌드 도구
* TinTin++ 실행 파일

---

## 빌드 방법

Visual Studio 2022 개발자 명령 프롬프트에서 실행합니다.

```bat
cd /d D:\Tintin_gui\1
build_msvc.bat
```

또는 소스가 있는 폴더로 이동한 뒤:

```bat
build_msvc.bat
```

---

## 실행 방법

빌드 후 생성된 실행 파일을 실행합니다.

```text
ktin.exe
```

TinTin++ 실행 파일과 `main.tin` 설정 파일이 필요한 경우, KTin 설정에서 경로를 맞춰 사용합니다.

---

## TinTin++ ticker 테스트 예시

2초마다 `누구` 명령 전송:

```tintin
#ticker {test_who} {누구} {2}
```

3초마다 `점수` 명령 전송:

```tintin
#ticker {test_score} {점수} {3}
```

중지:

```tintin
#unticker {test_who}
#unticker {test_score}
```

## 권장 제외 파일

GitHub 저장소나 릴리즈 압축에는 아래 파일을 포함하지 않는 것을 권장합니다.

```gitignore
*.bak
*.bak_*
*.old
*.tmp
*.obj
*.res
*.exe
*.pdb
*.ilk
*.log
```

예:

```text
constants.h.bak
```

이런 파일은 수정 과정에서 생긴 백업 파일이며 컴파일에 필요하지 않습니다.

## 알려진 사항

* 갈무리 보기창의 ANSI 색상 표시는 안정성 문제로 현재 비활성화되어 있습니다.
* 갈무리 로그 파일에는 ANSI 원문이 저장됩니다.
* 추후 갈무리 ANSI 보기는 RichEdit 서식 방식이 아니라 별도 GDI 렌더러 방식으로 구현하는 것이 권장됩니다.
* KTin은 현재 멀티탭 방식이 아니며, 여러 세션은 새 창 띄우기로 사용합니다.
* 폰트 파일 배포 시 반드시 라이선스를 확인해야 합니다.
