KTin safe buildfix16
====================

변경 내용:
- 자동 로그인 패턴 기능을 복원했습니다.
- 환경설정 일반 설정에 아이디/비밀번호/성공/실패 패턴 입력을 복원했습니다.
- 주소록 새 주소/수정 팝업에 주소별 아이디/비밀번호/성공/실패 패턴 입력을 복원했습니다.
- 패턴 검사는 접속 후 60초 동안만 동작합니다.
- 아이디/비밀번호 전송이 끝나거나 성공 패턴/실패 패턴을 만나면 60초 전이라도 검사를 중지합니다.
- 접속유지 명령은 자동 로그인 검사 중에는 보내지 않고, 검사 종료 후 동작합니다.
- 자동 재접속은 복원하지 않았습니다. 재접속은 TinTin++ 스크립트에서 처리하는 방향입니다.

주의:
- 이 기능은 화면을 계속 감시하지 않고 접속 직후 제한 시간 안에서만 검사합니다.
- 패턴은 정규식이 아니라 단순 포함 문자열 검사입니다.
